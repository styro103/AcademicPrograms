Things You'll Need

First, you'll want to make sure you have the necessary tools to run a JAVA program.

1. A Server
2. A Program to Compile and Run the JAVA Program (For this Tutorial We'll Use Putty)
3. A Program to Write the JAVA Program (For This Tutorial We'll Use Notepad++)

The links posted above are shortcuts from the official websites where you can download Putty and Notepad++ respectively. Both are completely safe and virus free.
